# 更新日志

## v0.7.0-beta - 2026-07-03

### 🖥 桌面端重大更新
- 新增基于 Go + Wails 的 MO Gallery Desktop 桌面管理端。
- 桌面端补齐概览、照片、故事、博客、存储、设置等核心管理工作流，侧边栏、国际化、页面组件与 Wails bindings 持续完善。
- 新增桌面端 AI 助手与 AI 图片生成工作流，支持 AI 模型配置、API Key 显示/隐藏、模型设置界面优化。
- 桌面端上传体验大幅增强：支持数码/胶片上传模式、完整上传参数、拖拽导入、上传页布局与 Web 端对齐。
- 强化桌面端登录状态：支持会话持久化与校验、认证失败处理、退出登录二次确认。

### 📸 上传与图片处理
- 上传参数组件重构为可复用结构，统一数码/胶片上传参数与压缩模式。
- 默认启用图片压缩，新增 WASM AVIF 压缩能力，并将 AVIF 编码迁移到 Web Worker，避免浏览器主线程卡顿。
- 优化上传进度展示：完善重试逻辑、进度报告、压缩后文件名、原始文件名到存储文件名的提示与 tooltip 展示。
- 修复上传路径重复、目标文件名初始化、上传完成后文件名展示等问题。
- 新增本地存储路径穿越保护，提升上传与存储安全性。

### ✍️ 故事编辑与富文本
- 支持保存 TipTap JSON 内容，提升富文本内容结构化存储能力。
- 改进故事编辑器草稿管理、新建故事草稿流程与粘贴上传占位符替换。
- 编辑故事时上传弹窗可自动关联当前故事，并在每次进入编辑页时要求确认粘贴上传设置。
- 抽离粘贴上传占位符工具逻辑，提升编辑器上传流程稳定性。

### 🖼 相册、图库与照片管理
- 新增精选图库页面。
- 新增照片图库可见性字段，并支持批量更新照片图库显示状态。
- 优化后台照片网格加载与交互：新增照片注册接口、懒加载网格、照片网格滑块性能优化。
- 默认后台照片按拍摄时间倒序排列。
- 胶片灯箱新增上一张/下一张导航。

### 🛠 后台体验与性能优化
- 后台偏好设置与会话缓存集中管理。
- 优化后台设置、友链页面、照片批量操作布局、滚动阴影、筛选折叠动画与 Cmd+K 提示。
- 调整批量照片操作为侧边面板/折叠面板布局，减少界面拥挤。
- 修复 `content-visibility` 导致的空白闪烁问题。

### 🐛 修复与基础设施
- 修复 Prisma 时间区扩展导致日期错误偏移的问题。
- Vercel 构建改用 webpack，提升构建兼容性。
- 清理无效代码、更新本地工具配置与环境变量示例。
